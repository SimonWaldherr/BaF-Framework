CSS3 Buttons and Forms

CSS3 Buttons and Forms helps you easily create beautiful buttons.

Buttons

The "buttons" can be created by adding class="baf" to any appropriate <a>, <button>, or <input> element. 

<a href="#" class="baf">Post comment</a>
New Comment (link)

Colored Buttons

Colored buttons can be created by adding the red, blue, or green class to your button.

Disabled Buttons

Buttons can also be disabled. The traditional style used in Google Calendar displays the button with a white background and greyed out text. Add the disabled class for this style. If you'd rather keep the grey background but dim it, you can add the disabled2 class. 

<a href="#" class="baf disabled">Today</a>
<a href="#" class="baf disabled2">Today</a>
Today Today

Smaller Buttons

Default buttons too big? Shrink the text-size of the button with the sm class, or the size of the entire button with the mini class. 

<a href="#" class="baf sm">Month</a>
<a href="#" class="baf mini">Month</a>
Month Month

Grouped buttons

Groups of buttons can be created by wrapping them in an element given a class of baf-group. 

<div class="baf-group">
    <a href="#" class="baf">Archive</a>
    <a href="#" class="baf">Spam</a>
    <a href="#" class="baf">Delete</a>
</div>
Archive Spam Delete

Mixed groups

Displaying a mixture of grouped and standalone buttons, as might be seen in a toolbar, can be done by adding another wrapping element with the class baf-container. 

<div class="actions baf-container">
    <a href="#" class="baf">Compose New</a>
    
    <div class="baf-group">
        <a href="#" class="baf">Archive</a>
        <a href="#" class="baf">Spam</a>
        <a href="#" class="baf">Delete</a>
    </div>
                        
    <div class="baf-group minor-group">
        <a href="#" class="baf">Move to</a>
        <a href="#" class="baf">Labels</a>
    </div>

    <a href="#" class="baf">More</a>
</div>

Radio Button Groups

Sometimes your button group should act like a series of radio buttons, in which selecting one button deselects the others. Add the toggle-buttons class to your button-group div to accomplish this. To set the currently selected button, use the checked class. 

<div class="baf-group toggle-buttons">
    <a href="#" class="baf">Day</a>
    <a href="#" class="baf">Week</a>
    <a href="#" class="baf checked">Month</a>
    <a href="#" class="baf">4 Days</a>
    <a href="#" class="baf">Agenda</a>
</div>

Buttons with icons only

Create a button with only an icon (no text) by adding the no-text class, along with one of the nested icon classes below. 


<a href="#" class="baf no-text"><i class="icon-refresh"></i></a>
<a href="#" class="baf no-text"><i class="icon-trash icon-white"></i></a>

Or group them together, just like Gmail's paginator.

<div class="baf-group">
    <a href="#" class="baf no-text"><i class="icon-chevron-left"></i></a>
    <a href="#" class="baf no-text"><i class="icon-chevron-right"></i></a>
<div>

You can even mix and match with the disabled state.

<div class="baf-group">
    <a href="#" class="baf no-text disabled2"><i class="icon-chevron-left"></i></a>
    <a href="#" class="baf no-text"><i class="icon-chevron-right"></i></a>
<div>

Mini paginator buttons

The new back/forward buttons inside input elements on Google Flights is pretty slick. You can create those too by adding the xs pg bw and xs pg fw classes. 

<div class="baf-group">
    <a href="#" class="baf xs-no-text"><i class="xs-icon-bw"></i></a>
    <a href="#" class="baf xs-no-text"><i class="xs-icon-fw"></i></a>
<div>

Buttons with text and icon

A range of icons can be added (only for links and buttons) by adding a class of icon and any one of the provided icon classes below. 

<div class="baf-group">
    <a href="#" class="baf"><i class="icon-search"><i>Search</a>
<div>

Button dropdowns

Create buttons with dropdown menus, just like Google Docs, easily. We'll use Twitter Bootstrap's javascript to make it happen. 

<div class="baf-group">
  <a class="baf dropdown-toggle" data-toggle="dropdown" href="#">
    Dropdown Menu <span class="caret"></span>
  </a>
  <ul class="dropdown-menu">
      <li><a href="#">Document</a></li>
      <li><a href="#">Presentation</a></li>
      <li><a href="#">Spreadsheet</a></li>
      <li><a href="#">Form</a></li>
  </ul>
</div>
